Array

Chunk

Создает массив элементов, разбитых на группы длиной size. Если array не может быть разделено равномерно, окончательный кусок будет оставшимися элементами.

Compact

Создает массив исключая значениями равные false, а именно false, null, 0, "", undefined, и NaN.

Concat

Создает новый массив, который включает в себя все переданные аргументы в виде массивов или значений.

Difference

Создает массив значений массива, не включенных в другие заданные массивы, используя SameValueZero для сравнений на равенство. Порядок и ссылки на значения результатов определяются первым массивом.
